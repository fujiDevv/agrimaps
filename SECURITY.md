# Security Policy

## Supported Versions

The following versions of the Agrimaps backend and forecasting microservice are currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of Agrimaps seriously. If you discover a security vulnerability within this project, please send an email to the project maintainers directly instead of creating a public issue. 

You can report vulnerabilities to:
- **Email:** security@agrimaps.local (Please replace with actual contact email if available)

**What to expect:**
- We will acknowledge receipt of your vulnerability report within 48 hours.
- We will send you regular updates about our progress in addressing the issue.
- If the vulnerability is confirmed, we will work on a patch and release it as quickly as possible. We kindly ask that you keep the vulnerability confidential until a fix has been published.

Thank you for helping keep Agrimaps secure!
